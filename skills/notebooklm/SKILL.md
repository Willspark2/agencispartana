# NotebookLM MCP

Research through your notebooks in NotebookLM.

## Configuration
Requires Google Login. Since this is a server, manual authentication may be needed via the `setup_auth` tool.

## Usage
Run `mcporter call notebooklm.ask_question question="your question"`

### Available Tools
- `ask_question`: Ask a question to NotebookLM
- `add_notebook`: Add a notebook link to the library
- `list_notebooks`: List all notebooks
- `setup_auth`: Start the authentication flow
